# Clicky-Game

## Overview 

This app is a memory game with React. The idea of the game is to click on the cards without clicking on the same one twice until you click all 12 characters.

![Screenshot](public/mario.png)

## Technologies used
- React
- Bootstrap
- CSS
- ES6
